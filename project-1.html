Krom

Krom is a synthetic textile maker that markets and designs tailor-made textile products for the Automotive, Railway, Nautical, Hospitality and Home sectors.

Project Overview

Krom was a big challenge, the site has a custom propiertary CMS, an eCommerce, forms, animations, and serves a different layout based on device width (Desktop or Mobile). The layout was fairly straigthfoward but has some litle neat things like parallax backgrounds, animations, searchbar. The hardest part was implementing the custom eCommerce code from another site without breaking it. I mostly managed but I still had to consult a senior developer here and there.
